Natura11y's text sizes and styles were developed to balance content density and reading comfort under typical usage conditions..

Banner Headline

The banner-headline utility is named after the large headline in a newspaper that stretches across the top of the front page. It is useful for styling a header larger (or visually different) than the typical <h1> heading.

Section Headings

Natura11y provides six levels of section headings, <h1> through <h6>. Styles for each level are consistent for a tag header (e.g. <h1>) and a utility class header (e.g. .h1)

Font Size Utilities

Change the font-size of text, buttons, navigation, or entire forms with Natura11y's font size utilities.
